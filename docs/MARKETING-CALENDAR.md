# VowNook — 90-Day Marketing Calendar (week-by-week)

Check the boxes. **Week 1 = your beta launch week** (dates are relative — fill in
the real ones). Strategy: `MARKETING-STRATEGY.md` · Copy: `LAUNCH-MARKETING.md`.

> **Reality check for a solo founder:** don't do everything. The two jobs that
> matter most every single week are **recruit real vendors** and **talk to real
> couples**. Everything else supports those.

---

## The weekly rhythm (repeat EVERY week — the habit loop)
Copy this block into each week; the dated items below are *on top* of it.
- [ ] **Supply:** 10–15 vendor outreach touches (DM/email) + reply to every interested one
- [ ] **Content:** review + publish the week's autopilot blog post; 5 Pinterest pins; 3 IG/TikTok posts
- [ ] **Community:** 3–5 genuinely helpful Reddit/FB-group replies (mention VowNook sparingly)
- [ ] **Talk to users:** 2–3 real conversations (a couple or a vendor) — note what you learn
- [ ] **Measure:** 15 min — GA4 traffic, Search Console (impressions/indexing), admin signups/bookings

---

# PHASE 1 — Prove the loop (Weeks 1–4)
**Goal:** real founding vendors live + real couples planning + first booking & review.

## Week 1 — Launch the beta
- [ ] Push + deploy; run the 3 seeders; finish the smoke-test (`DEPLOY-DAY.md`)
- [ ] Post the FOUNDING50 founder announcement (LinkedIn/personal) + email anyone you know
- [ ] Build the founding-vendor target list (40–60 vendors across metros/categories)
- [ ] Set up brand profiles: Pinterest, Instagram, TikTok, Facebook Page, Google Business, LinkedIn
- [ ] Invite the first 5 couple testers personally

## Week 2 — First vendors + first couples
- [ ] Send the founding-vendor pitch to the top 20 on your list
- [ ] Onboard the first 3–5 vendors who say yes (walk them through their listing)
- [ ] Build the Canva brand kit (pin + carousel + story templates)
- [ ] Confirm analytics fire (GA Realtime) + submit `sitemap.xml` in Search Console
- [ ] Collect feedback from week-1 testers; fix the top 3 friction points

## Week 3 — Tighten activation
- [ ] Watch 5 Clarity session recordings — where do couples get stuck? Fix it.
- [ ] Get to 10+ founding vendors live (keep outreach going)
- [ ] Ask each live vendor to post "find us on VowNook" to their audience
- [ ] Publish a "founding vendor spotlight" (their content = their followers see you)
- [ ] Request indexing in Search Console: homepage, blog, top category pages

## Week 4 — First booking push
- [ ] Personally help 2–3 testers actually request a quote / book a vendor
- [ ] Chase your **first real booking** → triggers your **first real review**
- [ ] Review the funnel: signups → activation % → inquiries → bookings. Find the leak.
- [ ] Turn on **SSR** (`INERTIA_SSR_ENABLED`) — ask me to verify it's rendering
- [ ] Phase-1 retro: what's working? Double down next phase.

# PHASE 2 — Density in 1–2 metros (Weeks 5–8)
**Goal:** Toronto + one more metro genuinely useful (enough vendors per category).

## Week 5 — Go deep on Toronto
- [ ] Map Toronto gaps: which categories have <3 real vendors? Target those.
- [ ] Turn on the **blog + local-SEO autopilots** (`--force` test each first)
- [ ] Start Pinterest in earnest (10 pins this week, themed boards)
- [ ] First PR outreach: pitch the "honest reviews tied to real bookings" story to 5 local/wedding outlets

## Week 6 — Second metro + community
- [ ] Pick metro #2 (Ottawa or Hamilton); start vendor outreach there
- [ ] Go deeper in 2 wedding communities (Reddit + an Ontario FB group) — be a regular, not a spammer
- [ ] Purge demo vendors in any category that now has 3+ real ones (`marketplace:demo --purge` when ready overall)
- [ ] Publish a city-specific piece (e.g., "wedding cost in Toronto vs Ottawa")

## Week 7 — Referral + reviews engine
- [ ] Make sure the referral loop + post-wedding review requests are firing (scheduler on)
- [ ] Ask happy testers to refer a friend (share their referral link)
- [ ] Get to 5+ real reviews live (they power trust + SEO)
- [ ] A/B your homepage hook if activation is below ~30%

## Week 8 — Measure + decide
- [ ] CAC + activation by channel: which is cheapest + best? Note it.
- [ ] Check Search Console: are category/blog pages getting impressions yet?
- [ ] Planner outreach: pitch Planner HQ to 3–5 Ontario planners (channel + segment)
- [ ] Phase-2 retro: pick the 1–2 channels to concentrate on in Phase 3

---

# PHASE 3 — Compound + expand (Weeks 9–13)
**Goal:** lean into the winner, loops self-sustaining, expand metros.

## Week 9 — Concentrate
- [ ] Double the effort on your best channel (from Week 8); cut the worst
- [ ] First small **paid test** (only if activation is solid): Pinterest or Google Search on one high-intent local term, hard budget cap
- [ ] Publish a "real numbers / data" piece you can pitch for backlinks

## Week 10 — Expand supply
- [ ] Add metro #3 vendor outreach
- [ ] Re-engage vendors who went quiet (the unanswered-inquiries nudge helps)
- [ ] Turn first reviews into social proof: testimonial graphics for IG/Pinterest

## Week 11 — Partnerships
- [ ] Line up 1–2 partnerships: a venue (refers couples) or a planner (channel)
- [ ] Guest content / a feature swap with a complementary local brand
- [ ] Review paid-test results: scale if it pays back, kill if not

## Week 12 — Optimize the funnel
- [ ] Fix the biggest drop-off in the funnel (from your weekly data)
- [ ] Push Atelier value: make sure FOUNDING50 couples see what they unlocked (→ word of mouth)
- [ ] Collect + publish 10+ reviews milestone

## Week 13 — Review the quarter, plan the next
- [ ] North-star check: are **weddings actively being planned** growing week over week?
- [ ] Full retro: best channel, CAC, bookings, GMV, reviews, what to drop
- [ ] Decide Q2: which metros to add, whether to scale paid, what to build next
- [ ] Write next 90-day plan from what actually worked

## How to use this
- Each Monday: copy the **weekly rhythm** + that week's dated tasks into your todo app.
- Each Friday: tick what's done, note the one number (weddings being planned), carry over the rest.
- Don't be a hero — **shipping 70% of this consistently beats 100% for two weeks then burnout.**

## When you're behind (and you will be some weeks)
Protect, in order: (1) talk to users, (2) vendor outreach, (3) publish content,
(4) community, (5) everything else. The top two never get skipped.
